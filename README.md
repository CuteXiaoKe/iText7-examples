## 目录结构

&nbsp;&nbsp;&nbsp;&nbsp;iText-start文件夹对应《iText 7 Jump-Start Tutorial》，iText-buildingblock文件夹对应《iText 7: Building Blocks》，里面每一目录都是IDEA工程，采用maven构建项目

## 运行

&nbsp;&nbsp;&nbsp;&nbsp;使用IDE打开工程，maven构建项目即可。
